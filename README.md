# ayo-ooo/homebrew-tap

Homebrew tap for [Ayo](https://github.com/ayo-ooo/ayo) CLI.

## Installation

```bash
brew tap ayo-ooo/tap
brew install ayo
```

## What is Ayo?

Ayo is a CLI tool for building AI agents with structured inputs and outputs. It generates standalone Go binaries that integrate with LLM providers.

See [github.com/ayo-ooo/ayo](https://github.com/ayo-ooo/ayo) for more information.
